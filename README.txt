                                                    WPFTOWIN
                                                    ========

                          Windows Presentation Foundation Brushes to Windows Form Brushes


Read Me
      
      This Tool is Mainly Designed for Converting WPF(XAML) Code to WIN(CS) Code for WPF LinearGradientBrush,
      RadialGradientBrush,SolidColorBrush to System.Drawing.Drawing2D.LinearGradientBrush,PathGradientBrush,SolidBrush.
